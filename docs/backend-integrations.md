# Backend Integrations

## Groq
- Used for roadmap text, explanations, role summaries, coaching copy.
- Fallback: deterministic templates if unavailable.

## PostgreSQL + Prisma
- Persistent storage for profiles, roles, skills, certifications, technologies, roadmaps, and feedback.

## Auth
- NextAuth chosen for fastest full-stack integration with Next.js.

## Storage
- Local file uploads for MVP admin ingestion; cloud abstraction later.
